fogbugz-timesheet
=================

A small library for updating the timesheet on FogBugz.

Installation
------------
Ubuntu:
```bash
sudo apt-get install ruby rubygems
sudo gem install bundler
sudo bundle install
```

RVM:
> gem install bundler
bundle install

Setup
--------------------------
> ruby ./bin/write_config.rb <fogbugz_url> <email> <password>
